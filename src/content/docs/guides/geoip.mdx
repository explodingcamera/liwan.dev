---
title: GeoIP Lookup
---

Liwan can optionally use the visitor's IP address during request processing to derive country and city. The IP address itself is not stored.

To enable GeoIP, either configure MaxMind credentials or provide a local MaxMind-compatible `.mmdb` file. See the [Configuration Reference](/reference/configuration) for all options.

MaxMind is the officially supported provider. If you use a different provider, make sure it exports a compatible `.mmdb` file.

GeoIP lookups use extra CPU and memory, which can matter on low-end hardware.

## MaxMind GeoIP Database

### Creating a MaxMind Account

For country and city lookups, the free offering from MaxMind is sufficient. You can sign up for a free account on the [MaxMind website](https://www.maxmind.com/geoip/signup).
Once you have created an account, you can go to the [Manage License Keys](https://www.maxmind.com/en/accounts/current/license-key) page to generate a license key and find your account ID.

### Example Configuration

To download the GeoIP database automatically, add:

```toml
# liwan.config.toml
[geoip]
maxmind_account_id="MY_ACCOUNT_ID"
maxmind_license_key="MY_LICENSE_KEY"
maxmind_edition="GeoLite2-City"
```

Replace `MY_ACCOUNT_ID` and `MY_LICENSE_KEY` with your MaxMind account ID and license key, respectively.

The next time you start Liwan, it will download the GeoIP database and periodically check for updates.

If you prefer to use a local GeoIP database file, you can specify the path to the file in the configuration:

```toml
# liwan.config.toml
[geoip]
maxmind_db_path="/path/to/GeoLite2-City.mmdb"
```

## Alternative Providers

You can use other GeoIP providers by keeping a local `.mmdb` file updated on disk.
Liwan automatically reloads the database file when it changes, so a cron job that periodically replaces the file is enough.

Set `geoip.maxmind_db_path` to the same destination path used by the script.

Some options you can evaluate:

- [IP Location DB](https://github.com/sapics/ip-location-db)
- [IPinfo Lite](https://ipinfo.io/lite)
- [IPLocate Free Databases](https://www.iplocate.io/free-databases)

When using a third-party source, verify update frequency, data quality, and license terms for your use case.

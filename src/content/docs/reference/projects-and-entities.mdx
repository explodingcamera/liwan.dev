title: Projects and Entities

import { Card } from "@astrojs/starlight/components";

Liwan uses **projects** and **entities** to organize analytics data.

<Card title="Projects">

Projects are dashboard containers. They group one or more entities for reporting and access control.

Depending on your setup, you might create one project per website, or one project that combines a website and related app.

</Card>

<Card title="Entities">

Entities are the individual websites or applications that send events. Each tracking snippet is tied to an entity.

Entities can belong to multiple projects, so the same source can appear in more than one dashboard.

</Card>
